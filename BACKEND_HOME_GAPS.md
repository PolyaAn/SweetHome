# Backend gaps for frontend section Home

----

## Статус

- Backend-разрывы для раздела `Дом` закрыты.
- Файл фиксирует результат проверки backend-контракта.
- Устаревшие требования к backend удалены.

----

## Закрытые backend-пункты

- Endpoint журнала событий доступен.
- Endpoints сценариев доступны.
- Endpoints автоматизаций доступны.
- Realtime-канал доступен.
- Realtime-события описаны.
- Проверка `entityId` при сохранении layout добавлена.
- Проверка `settingsJson` на валидный JSON добавлена.
- Ошибка удаления непустой комнаты добавлена.
